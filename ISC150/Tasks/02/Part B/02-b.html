Task 2 part B — a second way to create/update Web pages

  • Use "CyberDuck" to connect to moxie.cs.oswego.edu.
    (We'll go over how to do this in class.)

  • Once connected to moxie, navigate to your public_html directory and transfer your index.html to your Windows machine.

  • Open the file index.html on your Windows PC using WordPad. (Do NOT use NotePad.)

    Add the HTML markup show in bold to the page.
    (If you've managed to lose the other markup, add it as well.) 

    
<HTML>
  <HEAD>
    <TITLE>Home Page - alpha version 0.1</TITLE>
  </HEAD>
  <BODY>
    <DIV>
    <P>
    <h1>Home Page of YOUR NAME HERE</h1>
    </P>
    </DIV>

    <DIV>
    <P>
    <h2>My Courses</h2>
    </P>

    
    <A href="coursework/index.html">
    Index of courses
    </A>


        </DIV>

  </BODY>
</HTML>

  • Save the file on your Windows machine.

  • Use CyberDuck to transfer the file back to your public_html directory.

  • Test the results by typing:
    http://moxie.cs.oswego.edu/~YOUR-USER-ID/index.html
    into a browser address field.
    Note that there's a tilde before your user id in the URL.

  • You should now have a link on your home page that —
    when you hover over it shows something like:
    http://moxie.cs.oswego.edu/~YOUR-USER-ID/coursework/index.html
    at the bottom of the browser.
    and when you click on it, says
    Not Found

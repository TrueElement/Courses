Task 2 part A — one way to create/update Web pages

  • Use "putty" to get a secure login to moxie.cs.oswego.edu.

  • Once aboard moxie, type the following commands to:
    a) make sure you are in your home directory
    b) change into your public_html directory
    c) edit a file named index.html in that directory 

    cd
cd public_html
emacs -nw index.html

  • Type the following into the file index.html. 
    
<HTML>
  <HEAD>
    <TITLE>Home Page - alpha version 0.1</TITLE>
  </HEAD>
  <BODY>
    <DIV>
    <P>
    <h1>Home Page of YOUR NAME HERE</h1>
    </P>
    </DIV>

    <DIV>
    <P>
    <h2>My Courses</h2>
    </P>


    </DIV>

  </BODY>
</HTML>

  • Save the file by typing:
    Control-X Control-S
    Hold the control key down, don't let it up;
    push and release the 'X' key, then push and release the 'S' key;
    now let the control key up.

  • Exit the emacs editor by typing:
    Control-X Control-C

  • Use chmod to set permissions on the file index.html.
    chmod go+r index.html.

  • Test the results by typing:
    http://moxie.cs.oswego.edu/~YOUR-USER-ID/index.html
    into a browser address field.
    Note that there's a tilde before your user id in the URL.

  • Note that you can also type:
    http://moxie.cs.oswego.edu/~YOUR-USER-ID/
    without the index.html and it the loads the file named index.html. Why do you think this works?

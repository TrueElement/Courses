Task 1

  • Use "putty" to get a secure login to moxie.oswego.edu.

  • Use mkdir to create the directory public_html. (Directories are often called folders, especially in GUI environments.) 
    mkdir public_html

  • Use chmod to set permissions on the directory you've just created and on your home directory. Note the dot (period) at the end of the first line. (There's a space before the dot.) 
    chmod go+rx .
chmod go+rx public_html

  • Use echo to create a file named hello.html in the directory public_html. Note the single quotes around the hello world phrase. 
    cd public_html
echo '<h1>Hello World!</h1>' > hello.html

  • Use chmod to set permissions on the file hello.html. 
    chmod go+r hello.html

  • See if you can read the message by accessing the page in your Web browser. 
    http://moxie.cs.oswego.edu/~your-user-id/hello.html
